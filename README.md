# 🤚 MediaPipe 손가락 끝점 감지 예제

Google MediaPipe를 사용하여 웹에서 실시간으로 손가락 끝점을 감지하고 랜드마크를 시각화하는 예제입니다.

## ✨ 주요 기능

- 🎥 **실시간 웹캠 손 감지**: MediaPipe Hands 모델을 사용한 실시간 손 추적
- 👆 **검지 끝점 강조**: 검지(Index Finger) 끝점을 특별히 표시
- 🎯 **모든 랜드마크 시각화**: 손의 모든 21개 랜드마크를 실시간으로 표시
- 📊 **좌표 정보 표시**: 각 랜드마크의 X, Y, Z 좌표를 실시간으로 표시
- 🎨 **아름다운 UI**: 반응형 디자인과 직관적인 사용자 인터페이스

## 🚀 사용법

### 1. 웹 브라우저에서 실행

```bash
# 프로젝트 폴더로 이동
cd /Users/uhdmac/Documents/GitHub/hand-gesture-test

# 간단한 HTTP 서버 실행 (Python 3)
python3 -m http.server 8000

# 또는 Node.js가 설치되어 있다면
npx http-server -p 8000
```

### 2. 브라우저에서 접속

브라우저에서 `http://localhost:8000` 또는 `http://127.0.0.1:8000`으로 접속합니다.

### 3. 사용 방법

1. **시작 버튼 클릭**: 카메라 권한을 허용하고 웹캠을 시작합니다
2. **손 보이기**: 카메라에 손을 보여주면 자동으로 감지됩니다
3. **검지 끝점 확인**: 검지 끝점이 빨간색 원으로 강조 표시됩니다
4. **좌표 확인**: 오른쪽에 모든 랜드마크의 좌표가 실시간으로 표시됩니다

## 📋 랜드마크 정보

MediaPipe Hands는 각 손에 대해 **21개의 랜드마크**를 제공합니다:

- **손목**: 1개
- **엄지**: 4개 (CMC, MCP, IP, 끝점)
- **검지**: 4개 (MCP, PIP, DIP, 끝점) ⭐
- **중지**: 4개 (MCP, PIP, DIP, 끝점)
- **약지**: 4개 (MCP, PIP, DIP, 끝점)
- **새끼손가락**: 4개 (MCP, PIP, DIP, 끝점)

## 🎯 검지 끝점 (Index Finger Tip)

- **랜드마크 인덱스**: 8번
- **표시 방법**: 빨간색 원과 "검지 끝점" 텍스트로 강조
- **좌표**: X, Y, Z 3D 좌표 실시간 표시

## 🔧 문제 해결

### 카메라가 작동하지 않는 경우
1. HTTPS 환경에서 실행하거나 `localhost`에서 실행
2. 브라우저의 카메라 권한을 허용했는지 확인
3. 다른 애플리케이션에서 카메라를 사용하고 있지 않은지 확인

### 성능이 느린 경우
1. `modelComplexity` 값을 0으로 변경 (정확도는 낮아지지만 속도 향상)
2. `maxNumHands`를 1로 제한
3. 브라우저를 최신 버전으로 업데이트
